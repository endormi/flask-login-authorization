Flask-Login-Boilerplate

Simple user authorization and authentication.


This is a boilerplate for Flask login and sign up that you can use for your flask websites. It uses user authorization and authentication to display user specific content. I used flask_login, werkzeug.security, Flask-SQLAlchemy for this project.
